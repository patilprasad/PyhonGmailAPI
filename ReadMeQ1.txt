#Program to download emails using specific keyword in Subject.(Gmail API).

Library used "imaplib"---- https://docs.python.org/2/library/imaplib.html.

Make sure you go through this https://support.google.com/accounts/answer/6010255 and allow proper permission in GMail settings to support imaplib.

Run "Question1.py"

'Download_file.txt' file contains the emails based on the keyword choice. (Note: Each mail start with sender email in .txt file)

"*********" sequence marks separate mails in final file.

Attachments are named based on the directory and sender.
